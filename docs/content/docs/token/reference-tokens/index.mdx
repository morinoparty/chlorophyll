# Reference Tokens

Reference tokens are the raw values that define the foundation of the design system. They are not intended to be used directly in components, but rather serve as building blocks for semantic tokens.

## Available Tokens

- [Colors](/docs/theme/reference-tokens/colors) - Base color palette with 12 steps
- [Radii](/docs/theme/reference-tokens/radii) - Border radius values
- [Font Weights](/docs/theme/reference-tokens/font-weights) - Typography weight values
- [Line Heights](/docs/theme/reference-tokens/line-heights) - Line spacing values
- [Letter Spacings](/docs/theme/reference-tokens/letter-spacings) - Character spacing values
- [Opacity](/docs/theme/reference-tokens/opacity) - Transparency values
- [Durations](/docs/theme/reference-tokens/durations) - Animation timing values
- [Easings](/docs/theme/reference-tokens/easings) - Animation curve values
- [Aspect Ratios](/docs/theme/reference-tokens/aspect-ratios) - Width to height ratios
- [Border Widths](/docs/theme/reference-tokens/border-widths) - Border thickness values
