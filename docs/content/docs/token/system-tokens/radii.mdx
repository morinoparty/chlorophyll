# Semantic Radii

Semantic border radius tokens for consistent rounding.

These tokens map to the reference radii tokens but provide semantic naming for different use cases.

## Scale

| Token | Value | Usage |
|-------|-------|-------|
| `radii.none` | 0 | No rounding |
| `radii.sm` | 0.25rem | Small rounding |
| `radii.md` | 0.375rem | Medium rounding |
| `radii.lg` | 0.5rem | Large rounding |
| `radii.xl` | 0.75rem | Extra large rounding |
| `radii.2xl` | 1rem | 2x large rounding |
| `radii.full` | 9999px | Full rounding (circle/pill) |
