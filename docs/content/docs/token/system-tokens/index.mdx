# System Tokens

System tokens are semantic tokens that map to reference tokens based on their purpose. They provide a layer of abstraction that makes it easier to maintain consistency across the design system.

## Available Tokens

- [Colors](/docs/theme/system-tokens/colors) - Semantic color tokens (bg, fg, surface, etc.)
- [Borders](/docs/theme/system-tokens/borders) - Border style definitions
- [Typography](/docs/theme/system-tokens/typography) - Text style combinations
- [Z-Index](/docs/theme/system-tokens/z-index) - Stacking order values
- [Shadows](/docs/theme/system-tokens/shadows) - Box shadow definitions
- [Radii](/docs/theme/system-tokens/radii) - Semantic border radius
- [Spacing](/docs/theme/system-tokens/spacing) - Margin and padding values
- [Animations](/docs/theme/system-tokens/animations) - Animation keyframe definitions
