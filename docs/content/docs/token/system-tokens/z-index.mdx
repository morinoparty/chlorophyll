# Z-Index

Stacking order values for layered elements.

Z-index tokens ensure consistent layering across components like modals, dropdowns, and tooltips.

## Scale

| Token | Value | Usage |
|-------|-------|-------|
| `zIndex.hide` | -1 | Hidden elements |
| `zIndex.base` | 0 | Base layer |
| `zIndex.docked` | 10 | Docked elements |
| `zIndex.dropdown` | 1000 | Dropdowns |
| `zIndex.sticky` | 1100 | Sticky elements |
| `zIndex.banner` | 1200 | Banners |
| `zIndex.overlay` | 1300 | Overlays |
| `zIndex.modal` | 1400 | Modals |
| `zIndex.popover` | 1500 | Popovers |
| `zIndex.toast` | 1700 | Toasts |
| `zIndex.tooltip` | 1800 | Tooltips |
