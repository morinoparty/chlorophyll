# Animations

Animation keyframe definitions for motion.

Animation tokens provide consistent motion patterns across the design system.

## Available Animations

- `fadeIn` - Fade in from transparent
- `fadeOut` - Fade out to transparent
- `slideIn` - Slide in from offset
- `slideOut` - Slide out to offset
- `scaleIn` - Scale in from smaller
- `scaleOut` - Scale out to smaller

## Usage

```css
animation: var(--mpc-animations-fade-in);
```
