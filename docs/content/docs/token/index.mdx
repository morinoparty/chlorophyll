# Theme

Design tokens and theming for Chlorophyll.

Chlorophyll uses a token-based design system that provides consistent styling across all components.

## Token Types

### Reference Tokens

Reference tokens are the raw values that define the design system. They include:

- **Colors** - Base color palette with 12 steps
- **Radii** - Border radius values
- **Font Weights** - Typography weight values
- **Line Heights** - Line spacing values
- **Letter Spacings** - Character spacing values
- **Opacity** - Transparency values
- **Durations** - Animation timing values
- **Easings** - Animation curve values
- **Aspect Ratios** - Width to height ratios
- **Border Widths** - Border thickness values

### System Tokens

System tokens are semantic tokens that map to reference tokens based on their purpose:

- **Colors** - Semantic color tokens (bg, fg, surface, etc.)
- **Borders** - Border style definitions
- **Typography** - Text style combinations
- **Z-Index** - Stacking order values
- **Shadows** - Box shadow definitions
- **Radii** - Semantic border radius
- **Spacing** - Margin and padding values
- **Animations** - Animation keyframe definitions
