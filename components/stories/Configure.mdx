# 🌿 Chlorophyll 🌿

Chlorophyllは、モダンで使いやすいUIコンポーネントライブラリです。ReactとTypeScriptを使用して作られており、アクセシビリティと使いやすさを重視しています。

## 🎨 デザインシステム

Chlorophyllは、一貫性のあるデザインシステムを提供します。アプリケーション全体で統一されたデザインを実現します。

## 🛠 技術スタック

- 🎨 **言語**: TypeScript
- ⚛️ **フレームワーク**: React
- 🎯 **CSSライブラリ**: PandaCSS
- 🎭 **Headless UI**: React Aria

## 📦 インストール方法

```bash
pnpm i @moripa/chlorophyll
```

## 💻 使い方

```tsx
import { Button } from '@moripa/chlorophyll';

function App() {
  return (
    <Button>クリックしてね！</Button>
  );
}
```

### コンポーネントライブラリ
