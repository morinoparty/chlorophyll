# 🌿 Chlorophyll 🌿

Chlorophyll is a UI component library that is modern and easy to use. It is made with React and TypeScript and emphasizes accessibility and usability.


## 🎨 Design System

Chlorophyll provides a consistent design system. It realizes a unified design across the application.

## 🛠 Tech Stack

- 🎨 **Language**: TypeScript
- ⚛️ **Framework**: React
- 🎯 **CSS Library**: PandaCSS
- 🎭 **Headless Library**: React Aria

## 📦 Installation

```bash
pnpm i @moripa/chlorophyll
```

## 💻 Usage

```tsx
import { Button } from '@moripa/chlorophyll';

function App() {
  return (
    <Button>Click me!</Button>
  );
}
```
