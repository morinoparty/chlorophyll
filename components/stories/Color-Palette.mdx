# About Color Palette

This page displays the color palette used in the Chlorophyll design system.
Each color is available through the Panda CSS token system.

## Usage

Color tokens can be used as follows:

```tsx
// In a component
<div className={css({ backgroundColor: "green.500", color: "white" })} />

// In a style definition
const styles = {
  container: css({
    backgroundColor: 'blue.100',
    color: 'gray.800',
  })
}
```
